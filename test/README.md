Python 中 数组套字典的排序(用lambda实现)
dict = [
    {'id':'4','name':'b'},
    {'id':'6','name':'c'},
    {'id':'3','name':'a'},
    {'id':'1','name':'g'},
    {'id':'8','name':'f'}
]
排序后：[{'id': '1', 'name': 'g'}, {'id': '3', 'name': 'a'}, {'id': '4', 'name': 'b'}, {'id': '6', 'name': 'c'}, {'id': '8', 'name': 'f'}]
